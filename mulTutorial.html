Multiplication Tutorial

Multiplication is written using the multiplication sign, "x", to indicate the number on the left being multiplied by the number on the right.

Multiplication is when you take one number and add it together a number of times.

For example, if you have 2 apples and want to have 6 apples, you can multiply your current amount by 3. This is because 2 x 3 = 6, is the same as writing 2 + 2 + 2 = 6

  • Like the example on the right, 2 apples x 3 = 6 apples reads as, 2 apples multiplied by 3 apples equals 6 apples.
  • 1 x 0 = 0 reads, one multiplied by zero equals zero
  • 5 x 2 = 10 reads, five multiplied by two equals ten

Special cases for multiplication:

  • Any number multiplied by 0, equals 0
    Example: 5 x 0 = 0
  • Any number multiplied by 1, equals the same number.
    Example: 5 x 1 = 5
